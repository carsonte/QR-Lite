# QR Lite v1.0.0

首个正式版，聚焦三个目标：好上手、能处理真实物料、分发给同事后更稳。

## Highlights

- 新增双模式工作流
- `自动替换`：适合原图里本来就有二维码
- `手动放一个`：适合原图里没有二维码、只有空框或占位框
- 页面文案重写为更直白的操作指令，降低误操作

## Image Processing

- 支持上传二维码图片或直接输入内容生成二维码
- 支持裁掉二维码图片下方说明文字
- 支持拖拽蓝框微调位置和大小
- 支持 `CMYK JPEG` 原图替换 `RGB` 二维码后仍保持 `CMYK JPEG`
- 尽量保留 ICC Profile / DPI / EXIF

## Performance

- 大图自动识别先在缩小图上执行，再回推原图坐标
- 透视贴图改为局部区域混合，不再每次对整张大图做大范围运算
- 启动改为延后加载重模块，减少弱性能设备上的启动卡顿

## Stability

- 修复打包版启动卡住问题
- 修复关闭软件时报 `Tcl_AsyncDelete` 的问题
- 修复自动模式和手动模式切换后蓝框状态串线的问题
- 固定打包版服务协议，避免缺失 websocket 相关模块导致启动失败

## Packaging

- 当前正式分发方案为 `onedir`
- 正式产物位置：`dist/QRLite/QRLite.exe`
- 当前打包体积约 `159 MB`
- 分发给同事时，请发送整个 `dist/QRLite` 目录，而不是只发 `QRLite.exe`

## Notes

- 这是本地网页工具，不上传云端
- 正式包更适合放到 GitHub Releases，而不是直接提交到仓库历史
