# QR Lite v1.0.0

The first public release of QR Lite focuses on three things: fast everyday use, clearer UI copy, and safer output for real design workflows.

## Highlights

- Auto Replace mode for images that already contain a QR code
- Manual Placement mode for empty boxes, white boxes, and placeholders
- Clearer button labels and hints for non-technical teammates
- Better support for `CMYK JPEG` source images
- Large-image performance improvements for weaker laptops

## Image Processing

- Upload a replacement QR image or generate one from text
- Crop away extra text under an uploaded QR image
- Drag the blue box to fine-tune placement and size
- Preserve ICC profile / DPI / EXIF whenever possible
- Keep output as `CMYK JPEG` when replacing an `RGB` QR code inside a `CMYK JPEG` source image

## Stability

- Fixed the packaged startup freeze issue
- Fixed shutdown errors such as `Tcl_AsyncDelete`
- Fixed mode-switch state leakage between auto and manual placement
- Locked packaged startup to the non-websocket server path to avoid missing-module launch failures

## Packaging

- Official distribution format: `onedir`
- Packaged app path: `dist/QRLite/QRLite.exe`
- Recommended teammate distribution: zip the whole `dist/QRLite` folder
